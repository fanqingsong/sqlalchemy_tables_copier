============
Installation
============


Stable release
--------------

To install SQLAlchemy Tables Copier, run this command in your terminal:

.. code-block:: console

    $ pip install sqlalchemy_tables_copier

This is the preferred method to install SQLAlchemy Tables Copier, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for SQLAlchemy Tables Copier can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/fanqingsong/sqlalchemy_tables_copier

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/fanqingsong/sqlalchemy_tables_copier/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/fanqingsong/sqlalchemy_tables_copier
.. _tarball: https://github.com/fanqingsong/sqlalchemy_tables_copier/tarball/master
